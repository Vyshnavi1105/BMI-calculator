# BMI-calculator
Developed a Python-based BMI Calculator that takes user input for name, weight, and height, calculates BMI using a standard formula, and classifies health status into underweight, normal, overweight, and obesity categories with personalized output.

# BMI Calculator

A simple BMI (Body Mass Index) Calculator project.

## Features
- Calculates BMI based on height and weight
- Easy to use
- Beginner-friendly project

## Technologies Used
- Python / HTML / CSS / JavaScript

## How to Run
1. Clone the repository
2. Run the file or open index.html

## Output
Shows BMI value and health category.
